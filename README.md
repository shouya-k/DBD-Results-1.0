# DBD Mobile Results

## アプリの概要 / Overview

DBD Mobile の戦績を登録できるアプリです。戦績登録から対戦キラー、得点、使用パーク、脱出人数を登録する事で直近戦績と各キラーの対戦戦績を記録することが出来ます。
